# 🧠 NLP Nexus



NLP Nexus is a powerful Natural Language Processing (NLP) web application designed to streamline text analysis and document summarization. It enables users to generate summaries, perform sentiment analysis, and visualize textual insights with ease.

Built using \*\*Flask + SpaCy + NLTK + TextBlob + Scikit-Learn\*\*, this project demonstrates applied NLP concepts in a practical, interactive format.

## 🚀 Features

### 📌 Text Summarization

Generate concise summaries from large documents or raw text by extracting the most relevant information.

### 😊 Sentiment Analysis

Analyze text to determine whether the sentiment is:



* Positive
* Negative
* Neutral



### 📂 Document Management

Upload and manage documents (including PDFs) for structured processing and analysis.



### ☁️ Word Cloud Generation

Visualize frequently occurring words to quickly identify dominant themes in your text.
